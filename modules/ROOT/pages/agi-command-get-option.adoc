[[agi-command-get-option]]
= `GET OPTION`

[source]
----
GET OPTION Dateiname Escape_Ziffern [Timeout]
----

Verhält sich wie `STREAM FILE`. Bietet aber zusätzlich
die Möglichkeit, einen _`Timeout`_ in Sekunden
anzugeben.

Ergebniswerte nach Ausführung:

[cols="2*"]
|===
| _`Wert`_ | DTMF-Eingaben
|===

.Asterisk-Versionen:
————————| *1.2* |————————| *1.4* |————————| *1.6* |————————

.Interner Hilfetext zu diesem AGI-Befehl in Asterisk 1.4:
Usage: GET OPTION <filename> <escape_digits> [timeout]
Behaves similar to STREAM FILE but used with a timeout option.

.Differenz des internen Hilfetexts von Asterisk 1.2 zu 1.4:
— keine —

[[functions-queue_member_count]]
= `QUEUE_MEMBER_COUNT()`

[source]
----
QUEUE_MEMBER_COUNT(warteschlange)
----

Gibt die Anzahl der Agenten (im Gegensatz zu den Anrufern) auf der
Warteschlange zurück.

[source]
----
; Anzahl der Agenten in der Warteschlange "support":
exten => 123,1,Set(foo=${QUEUE_MEMBER_COUNT(support)}) 
----

.Asterisk-Versionen:
| | | *1.4* |————————| *1.6* |————————

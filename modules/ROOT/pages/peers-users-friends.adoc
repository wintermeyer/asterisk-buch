= Peers, Users und Friends

Die Unterscheidung der Begriffe _Peer_,
_User_ und _Friend_ ist in der
Asterisk-Dokumentation nicht immer leicht verständlich dargestellt und
wird daher auch teilweise falsch auf manchen Webseiten wiedergegeben. Die
folgende Tabelle zeigt die jeweiligen Funktionen:

[cols="3*"]
|===
| Asterisk | <= | User
| Asterisk | => | Peer
| Asterisk | <=> | Friend
|===

Ein Peer kann also nur angerufen werden, ein User kann nur anrufen,
und ein Friend kann beides.

[TIP]
====
In der Praxis wird meistens nur _`Friend`_
benutzt. Entsprechend muss man sich diese Tabelle nicht einprägen,
sondern kann sie im Bedarfsfall nachschlagen.
====

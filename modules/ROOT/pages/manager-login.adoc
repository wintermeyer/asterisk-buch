[[manager-login]]
= `Login`

.Asterisk-Versionen:
| | | | | *1.6* |————————

.Interner Hilfetext zu diesem AMI-Befehl in Asterisk 1.6:
Action: Login
Synopsis: Login Manager
Privilege: <none>

.Achtung
Der interne Hilfetext ist unvollständig. Er sollte etwa so
lauten:Action: Login
Synopsis: Login Manager
Privilege: <none>
Description: Login to a manager session
Variables: (Names marked with * are required)
*Username: <username>
*Secret: <password>
Events: <on|off>

.Differenz des internen Hilfetexts von Asterisk 1.4 zu 1.6:
— in Asterisk 1.4 nicht vorhanden —

.Achtung
Diesen Befehl gibt es auch bereits in Asterisk 1.4, obwohl er da
nicht dokumentiert ist.

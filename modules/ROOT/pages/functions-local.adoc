[[functions-local]]
= `LOCAL()`

[source]
----
LOCAL(variablenname)
----

Setzt eine lokale Variabe, die nur im lokalen Gültigkeitsbereich
des Gosub-Stacks definiert ist.

[source]
----
macro beispiel( x ) {
    Verbose(1,x ist ${x});
    LOCAL(y)=x+x;
    Verbose(1,y ist ${y});
}
context default {
    123 => {
        &beispiel(4);
        // y wird zwar im Makro gesetzt, ist
        // aber hier nicht definiert
        Verbose(1,y ist ${y});
    }
}
----

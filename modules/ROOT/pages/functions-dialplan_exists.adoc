[[functions-dialplan_exists]]
= `DIALPLAN_EXISTS()`

[source]
----
DIALPLAN_EXISTS(context[,extension[,priority]])
----

Prüft, ob ein Context, eine Extension oder eine Priorität in einer
Extension im Dialplan vorhanden ist, und gibt dementsprechend
`1` oder `0` zurück.

[source]
----
exten => 123,1,Verbose(1,default     : DIALPLAN_EXISTS(default))
exten => 123,n,Verbose(1,123@default : DIALPLAN_EXISTS(default,123))
exten => 123,n,Verbose(1,intern      : DIALPLAN_EXISTS(intern))
exten => 123,n,Verbose(1,456@intern  : DIALPLAN_EXISTS(intern,456))
----

.Asterisk-Versionen:
| | | | | *1.6* |————————

.Interner Hilfetext zu dieser Funktion in Asterisk 1.6:
-= Info about function 'DIALPLAN_EXISTS' =-

[Syntax]
DIALPLAN_EXISTS(context[,extension[,priority]])

[Synopsis]
Checks the existence of a dialplan target.

[Description]
This function returns 1 if the target exits. Otherwise, it returns 0.

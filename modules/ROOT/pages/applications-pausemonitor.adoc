[[applications-pausemonitor]]
= `PauseMonitor()`

Stoppt das Monitoring eines Channels.

[source]
----
PauseMonitor()
----

Setzt die Aufzeichnung eines Channels vorübergehend aus, bis die
Pause mit `UnpauseMonitor()` wieder aufgehoben wird.

.Asterisk-Versionen:
| | | *1.4* |————————| *1.6* |————————

[Description]

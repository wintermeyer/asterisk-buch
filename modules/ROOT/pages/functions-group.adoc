[[functions-group]]
= `GROUP()`

[source]
----
GROUP([Kategorie])
----

Liest/setzt die Gruppe des Kanals (Kanäle lassen sich beliebig
gruppieren).

[source]
----
exten => 123,1,Set(GROUP()=ausgehende)                ; Gruppe setzen
exten => 123,n,GotoIf($[${GROUP_COUNT()} > 10]?200)   ; zu viele?
exten => 123,n,Dial(0123456)                          ; wählen
exten => 123,200,Set(DIALSTATUS=CHANUNAVAIL)       ; ablehnen
----

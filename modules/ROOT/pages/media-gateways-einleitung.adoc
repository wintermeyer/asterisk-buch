[[media-gateways-einleitung]]
= Einleitung

Sowohl für analoge als auch für ISDN-Anschlüsse kann man nicht nur
entsprechende Karten, sondern auch Media Gateways benutzen. Ein Media
Gateway konvertiert dann einen ISDN-Anschluss in mehrere SIP-Accounts und
umgekehrt. Es handelt sich dabei meistens um Blackbox-Lösungen, die über
eine Web-GUI oder über SSH konfiguriert werden.

Bei rein analogen Media Gateways spricht man oft auch von ATAs
(Analog-Telefon-Adapter), siehe auch xref:ata.adoc[Abschnitt 11.2, „ATA (Analog-Telefon-Adapter)“].

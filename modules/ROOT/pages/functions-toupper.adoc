[[functions-toupper]]
= `TOUPPER()`

[source]
----
TOUPPER(String)
----

Wandelt die mit dem Parameter übergebene Zeichenkette in
Großbuchstaben um und gibt diese zurück.

[source]
----
; Wandle den Namen des Anrufers in Großbuchstaben um
exten => 123,1,Answer()
exten => 123,n,Set(Foo=${TOUPPER(${CALLERID(name)})})
exten => 123,n,Hangup()
	
----

.Asterisk-Versionen:
| | | | | *1.6* |————————

.Interner Hilfetext zu dieser Funktion in Asterisk 1.6:
-= Info about function 'TOUPPER' =-

[Syntax]
TOUPPER(<string>)

[Synopsis]
Convert the string to upper case.

[Description]
Example: ${TOUPPER(Example)} returns "EXAMPLE"

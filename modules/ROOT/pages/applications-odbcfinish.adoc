[[applications-odbcfinish]]
= `ODBCFinish()`

Schließt das Result-Set einer ODBC-Abfrage.

[source]
----
ODBCFinish(ResultSetID)
----

Schließt das Result-Set (also die Ergebnismenge) einer ODBC-Abfrage
(siehe xref:functions-odbc_fetch.adoc[Abschnitt D.68, „`ODBC_FETCH()`“]), die mehrere Zeilen
zurückliefert.

.Asterisk-Versionen:
| | | | | *1.6* |————————

.Interner Hilfetext zu dieser Applikation in Asterisk 1.6:
-= Info about application 'ODBCFinish' =-

[Synopsis]
Clear the resultset of a successful multirow query

[Description]
ODBCFinish(<result-id>)
Clears any remaining rows of the specified resultset

.Differenz des internen Hilfetexts von Asterisk 1.4 zu 1.6:
— in Asterisk 1.4 nicht vorhanden —

.Siehe auch
xref:functions-odbc_fetch.adoc[Abschnitt D.68, „`ODBC_FETCH()`“]

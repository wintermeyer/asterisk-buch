[[functions-base64_encode]]
= `BASE64_ENCODE()`

[source]
----
BASE64_ENCODE(String)
----

Kodiert einen String mit Base64footnote:[http://de.wikipedia.org/wiki/Base64].

[source]
----
exten => 123,1,Set(foo=${BASE64_ENCODE("Hallo Welt")})
----

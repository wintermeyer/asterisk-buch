[[applications-proceeding]]
= `Proceeding()`

Signalisiert Fortschritt.

[source]
----
Proceeding()
----

Schickt eine Fortschrittsmeldung an den anrufenden Kanal.

.Asterisk-Versionen:
| | | | | *1.6* |————————

.Interner Hilfetext zu dieser Applikation in Asterisk 1.6:
-= Info about application 'Proceeding' =-

[Synopsis]
Indicate proceeding

[Description]
Proceeding(): This application will request that a proceeding message
be provided to the calling channel.

.Differenz des internen Hilfetexts von Asterisk 1.4 zu 1.6:
— in Asterisk 1.4 nicht vorhanden —

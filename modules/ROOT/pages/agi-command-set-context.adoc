[[agi-command-set-context]]
= `SET CONTEXT`

[source]
----
SET CONTEXT Contextname
----

Setzt den Context, der nach dem Ausführen der AGI-Applikation
angesprungen wird.

Ergebniswerte nach Ausführung:

[cols="2*"]
|===
| 0 | Gibt immer 0 zurück.
|===

[IMPORTANT]
====
Es wird nicht überprüft, ob der Context existiert. Wenn Sie einen
ungültigen Context angeben, wird der Anruf nach Beendigung des Skripts
aufgelegt.
====

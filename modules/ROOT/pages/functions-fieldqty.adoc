[[functions-fieldqty]]
= `FIELDQTY()`

[source]
----
FIELDQTY(Variablenname,Trennzeichen)
----

Gibt zurück, wie viele Teilstrings entstehen würden, wenn man den
Inhalt der Variablen anhand des Trennzeichens aufteilt.

[source]
----
exten => 123,1,Set(Var=hallo#du#da#am#telefon)
exten => 123,n,Set(Anzahl=${FIELDQTY(Var,#)})    ; Anzahl ist 5
----

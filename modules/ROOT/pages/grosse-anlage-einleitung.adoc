[[grosse-anlage-einleitung]]
= Einleitung

Nachdem Sie in xref:kapitel-hello-world.adoc[Kapitel 2, _„Hello World“_] gesehen haben,
wie man schnell und einfach eine kleine Asterisk-Telefonanlage aufsetzen
kann, werden im folgenden Kapitel am Beispiel einer fiktiven Firma die
umfangreichen Möglichkeiten und Funktionalitäten von Asterisk aufgezeigt
und beschrieben.

Jede Firma und damit jede Telefonanlage ist anders. Dieses Kapitel
soll deshalb als Anregung und Beispiel dienen. Die meisten Firmen werden
bei einer entsprechenden Anlage auf das fertige Toolkit
_Gemeinschaft_footnote:[http://www.amooma.de/gemeinschaft/] setzen.

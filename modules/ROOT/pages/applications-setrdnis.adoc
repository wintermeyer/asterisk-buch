[[applications-setrdnis]]
= `SetRDNIS()`

Setzt die RDNIS-Nummer.

[source]
----
SetRDNIS(Nummer)
----

[IMPORTANT]
====
`SetRDNIS()` ist ab Asterisk 1.4 nicht mehr
vorhanden und wurde durch die Funktion `CALLERID()`
(xref:functions-callerid.adoc[Abschnitt D.8, „`CALLERID()`“]) ersetzt:

[source]
----
Set(CALLERID(rdnis)=Nummer)
----
====

.Asterisk-Versionen:
————————| *1.2* | | | | |

.Interner Hilfetext zu dieser Applikation in Asterisk 1.2:
-= Info about application 'SetRDNIS' =-

[Synopsis]
Set RDNIS Number

[Description]
SetRDNIS(cnum): Set RDNIS Number on a call to a new
value.
SetRDNIS has been deprecated in favor of the function
CALLERID(rdnis)

.Differenz des internen Hilfetexts von Asterisk 1.2 zu 1.4:
— in Asterisk 1.4 nicht vorhanden —

[[applications-stopmusiconhold]]
= `StopMusicOnHold()`

Stoppt das Abspielen von Wartemusik.

[source]
----
StopMusicOnHold(param)
----

Stoppt die Wartemusik, die mit
`StartMusicOnHold()` gestartet wurde.

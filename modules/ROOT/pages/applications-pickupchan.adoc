[[applications-pickupchan]]
= `PickupChan()`

Einen Anruf entgegennehmen, der sich an einen anderen Kanal
richtet.

[source]
----
PickupChan(channel[&channel[...]])
----

Mit `Pickup()` kann ein Gesprächübernommen werden, das
an einem anderen Channel auf Entgegennahme wartet.
`PickupChan()` arbeitet also ähnlich wie
`Pickup()`, aber als Parameter wird statt einer Extension
ein Kanal angegeben.

.Asterisk-Versionen:
| | | | | *1.6* |————————

.Interner Hilfetext zu dieser Applikation in Asterisk 1.6:
-= Info about application 'PickupChan' =-

[Synopsis]
Pickup a ringing channel

[Description]
PickupChan(channel[&channel...]): This application can pickup any ringing channel

.Differenz des internen Hilfetexts von Asterisk 1.4 zu 1.6:
— in Asterisk 1.4 nicht vorhanden —

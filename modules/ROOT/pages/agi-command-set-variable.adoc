[[agi-command-set-variable]]
= `SET VARIABLE`

[source]
----
SET VARIABLE Variablenname Wert
----

Setzt die Kanal-Variable _`Variablenname`_ auf
den Wert _`Wert`_.

Ergebniswerte nach Ausführung:

[cols="2*"]
|===
| 1 | Erfolgreich (Variable wurde gesetzt)
|===

.Asterisk-Versionen:
————————| *1.2* |————————| *1.4* |————————| *1.6* |————————

.Interner Hilfetext zu diesem AGI-Befehl in Asterisk 1.4:
Usage: SET VARIABLE <variablename> <value>

.Differenz des internen Hilfetexts von Asterisk 1.2 zu 1.4:
— keine —

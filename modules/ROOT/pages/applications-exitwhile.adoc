[[applications-exitwhile]]
= `ExitWhile()`

Beendet innerhalb einer While-Schleife deren Ausführung.

[source]
----
ExitWhile()
----

Mit `ExitWhile()` kann innerhalb einer While-Schleife
die weitere Ausführung abgebrochen werden, unabhängig davon, ob die
While-Bedingung erfüllt ist oder nicht.

.Asterisk-Versionen:
| | | *1.4* |————————| *1.6* |————————

-= Info about application 'ExitWhile' =-

End a While loop

[Description]

Exits a While loop, whether or not the conditional has been satisfied.

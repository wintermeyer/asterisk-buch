Das Asterisk Gateway Interface (AGI) ist vergleichbar mit CGI auf
Webservern: externe Programme, die aus dem Dialplan heraus aufgerufen
werden, mit Asterisk kommunizieren (stdin/stdout) und Befehle
ausführen. AGI-Skripte lassen sich in praktisch jeder Sprache
schreiben, die Unix-Sockets und Zeilen-I/O kann.

[NOTE]
====
AGI ist weiterhin voll funktional und für kleine, synchrone
Dialplan-Aufrufe ("frage eine Datenbank ab, schreibe eine
Variable, mach weiter") immer noch das einfachste Werkzeug. Für
langlebige, Event-getriebene Anwendungen (IVR, Callcenter-Logik,
Integrationen) ist heute *xref:ari.adoc[ARI]* die bessere
Wahl.
====

Die vollständige Referenz aller AGI-Befehle finden Sie im Anhang
xref:agi-command-referenz.adoc[AGI-Befehle] sowie im CLI unter
`agi show commands`.

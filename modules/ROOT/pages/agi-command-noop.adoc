[[agi-command-noop]]
= `NOOP`

[source]
----
NOOP [Text]
----

Es wird nichts gemacht (`NOOP` steht für No-Operation).
_`Text`_ wird auf dem CLI ausgegeben.

Ergebniswerte nach Ausführung:

[cols="2*"]
|===
| 0 | Gibt immer 0 zurück.
|===

.Asterisk-Versionen:
————————| *1.2* |————————| *1.4* |————————| *1.6* |————————

.Interner Hilfetext zu diesem AGI-Befehl in Asterisk 1.4:
Usage: NoOp
Does nothing.

.Differenz des internen Hilfetexts von Asterisk 1.2 zu 1.4:
— keine —

.Differenz des internen Hilfetexts von Asterisk 1.4 zu 1.6:
--- in Asterisk 1.4
+++ in Asterisk 1.6
@@ -1,2 +1,3 @@
Usage: NoOp
Does nothing.
+ Runs Dead : Yes

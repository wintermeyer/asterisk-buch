[[functions-musicclass]]
= `MUSICCLASS()`

[source]
----
MUSICCLASS(Klasse)
----

Liest/setzt die Music-on-Hold-Klasse (Klasse der
Wartemusik).

[source]
----
; abfragen:
exten => 123,1,Set(foo=${MUSICCLASS()})

; auf "HeavyMetal" setzen:
exten => 123,1,Set(MUSICCLASS()=HeavyMetal)
----

[IMPORTANT]
====
Ab Asterisk 1.4 deprecated. Stattdessen
+Set(CHANNEL(musicclass)=_`default`_)+
verwenden, siehe xref:functions-channel.adoc[Abschnitt D.11, „`CHANNEL()`“].
====

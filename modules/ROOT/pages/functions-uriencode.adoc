[[functions-uriencode]]
= `URIENCODE()`

[source]
----
URIENCODE(String)
----

URI-kodiertfootnote:[http://de.wikipedia.org/wiki/URL_Encoding] einen String, d. h., dass Zeichen, die nicht in einer URI
vorkommen dürfen, durch Escape-Sequenzen der Form
+%_`XX`_+ ersetzt werden, wobei
`XX` der hexadezimale Wert des Zeichens ist.

[source]
----
; "Hallo Welt" kodieren:
exten => 123,1,Set(foo=${URIENCODE("Hallo Welt")})
; ergibt "Hallo%20Welt"
----

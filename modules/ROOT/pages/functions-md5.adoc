[[functions-md5]]
= `MD5()`

[source]
----
MD5(String)
----

Berechnet den MD5-Hash (Digest, Prüfsumme) eines Strings
(hexadezimale Rückgabe).

[source]
----
exten => 123,1,Set(foo=${MD5(${string})})
----

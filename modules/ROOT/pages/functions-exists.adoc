[[functions-exists]]
= `EXISTS()`

[source]
----
EXISTS(Variable)
----

Prüft, ob eine Variable definiert ist. Gibt 1 oder 0
zurück.

[source]
----
exten => 123,1,Set(Var1=test)
exten => 123,n,Set(Var2=)
exten => 123,n,Set(foo=${EXISTS(${Var1})})   ; foo ist 1
exten => 123,n,Set(foo=${EXISTS(${Var2})})   ; foo ist 0
----

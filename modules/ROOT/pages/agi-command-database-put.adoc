[[agi-command-database-put]]
= `DATABASE PUT`

[source]
----
DATABASE PUT Familie Schlüssel Wert
----

Speichert den Wert _`Wert`_ in der
Asterisk-Datenbank unter
_`Familie`_/_`Schlüssel`_ ab. Das
kann ein Hinzufügen oder ein Update sein.

Ergebniswerte nach Ausführung:

[cols="2*"]
|===
| 0 | Fehler
| 1 | Erfolgreich
|===

.Asterisk-Versionen:
————————| *1.2* |————————| *1.4* |————————| *1.6* |————————

.Interner Hilfetext zu diesem AGI-Befehl in Asterisk 1.4:
Usage: DATABASE PUT <family> <key> <value>
Adds or updates an entry in the Asterisk database for a
given family, key, and value.
Returns 1 if successful, 0 otherwise.

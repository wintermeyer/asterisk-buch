[[agi-command-database-deltree]]
= `DATABASE DELTREE`

[source]
----
DATABASE DELTREE Familie
----

Löscht eine gesamte _`Familie`_ aus der
Asterisk-Datenbank.

Ergebniswerte nach Ausführung:

[cols="2*"]
|===
| 0 | Fehler
| 1 | Erfolgreich
|===

.Asterisk-Versionen:
————————| *1.2* |————————| *1.4* |————————| *1.6* |————————

.Interner Hilfetext zu diesem AGI-Befehl in Asterisk 1.4:
Usage: DATABASE DELTREE <family> [keytree]
Deletes a family or specific keytree within a family
in the Asterisk database.
Returns 1 if successful, 0 otherwise.

.Differenz des internen Hilfetexts von Asterisk 1.2 zu 1.4:

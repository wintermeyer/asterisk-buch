[[functions-isnull]]
= `ISNULL()`

[source]
----
ISNULL(Wert)
----

Gibt 1 zurück, wenn der Wert NULL (kein Wert) ist, sonst
0.

[source]
----
exten => 123,1,Set(foo=${ISNULL(${Var1})})
----

[[functions-group_list]]
= `GROUP_LIST()`

[source]
----
GROUP_LIST()
----

Gibt eine durch Leerzeichen getrennte Liste aller Gruppen zurück,
die für den aktuellen Kanal gesetzt sind.

[source]
----
exten => 123,1,Set(foo=${GROUP_LIST()})
----

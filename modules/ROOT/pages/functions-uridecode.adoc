[[functions-uridecode]]
= `URIDECODE()`

[source]
----
URIDECODE(String)
----

Dekodiert einen URI-kodiertenfootnote:[http://de.wikipedia.org/wiki/URL_Encoding] String. Siehe `URIENCODE()`.

[source]
----
; "www.beispiel.de/?seite=Hallo%20Welt" dekodieren:
exten => 123,1,Set(foo=${URIDECODE("Hallo%20Welt")})
; ergibt "Hallo Welt"
----

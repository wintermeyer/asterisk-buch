[[functions-queue_member_list]]
= `QUEUE_MEMBER_LIST()`

[source]
----
QUEUE_MEMBER_LIST(warteschlange)
----

Gibt eine durch Komma getrennte Liste der Agenten auf einer
Warteschleife zurück.

[source]
----
; Agenten in der Warteschlange "support":
exten => 123,1,Set(foo=${QUEUE_MEMBER_LIST(support)})
; gibt z. B. 5,8,33 zurück
----

.Asterisk-Versionen:
| | | *1.4* |————————| *1.6* |————————

.Interner Hilfetext zu dieser Funktion in Asterisk 1.4:
-= Info about function 'QUEUE_MEMBER_LIST' =-

[Syntax]
QUEUE_MEMBER_LIST(<queuename>)

[Synopsis]
Returns a list of interfaces on a queue

[Description]
Returns a comma-separated list of members associated with the specified queue.

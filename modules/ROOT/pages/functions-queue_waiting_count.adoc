[[functions-queue_waiting_count]]
= `QUEUE_WAITING_COUNT()`

[source]
----
QUEUE_WAITING_COUNT(warteschlange)
----

Gibt die Anzahl der in einer Warteschlange wartenden Anrufer
zurück.

[source]
----
exten => 123,1,Set(count=${QUEUE_WAITING_COUNT(support)})
exten => 123,n,Verbose(1,In der Support-Queue warten ${count} Anrufer)
----

.Asterisk-Versionen:
| | | *1.4* |————————| *1.6* |————————

.Interner Hilfetext zu dieser Funktion in Asterisk 1.4:
-= Info about function 'QUEUE_WAITING_COUNT' =-

[Syntax]
QUEUE_WAITING_COUNT(<queuename>)

[Synopsis]
Count number of calls currently waiting in a queue

[Description]
Returns the number of callers currently waiting in the specified queue.

[[functions-quote]]
= `QUOTE()`

[source]
----
QUOTE(String)
----

Schließt einen String in Anführungszeichen
(„`"`“) ein und maskiert Anführungszeichen innerhalb
des Strings durch ein Escape-Zeichen
(„`\`“).

[source]
----
; wenn ${var} >>Die "Asterisk"-PBX<< ist
exten => 123,1,Set(foo=${QUOTE(${var})})
; dann wird >>"Die \"Asterisk\"-PBX"<< zurückggeben
----

.Asterisk-Versionen:

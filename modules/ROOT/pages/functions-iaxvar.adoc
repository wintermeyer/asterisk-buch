[[functions-iaxvar]]
= `IAXVAR()`

[source]
----
IAXVAR(variablenname)
----

Liest oder setzt eine Variable im IAX-Protokoll.

[source]
----
; auf dem einen Server eine Variable setzen:
exten => 123,1,Set(IAXVAR(beispiel)=Hallo)
exten => 123,n,Dial(IAX/456@andererserver)

; auf dem anderen Server die Variable auslesen:
exten => 456,1,Verbose(1,beispiel: ${IAXVAR(beispiel)})
exten => 456,n,Hangup()
----

.Asterisk-Versionen:
| | | | | *1.6* |————————

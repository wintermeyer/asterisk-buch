[[functions-curl]]
= `CURL()`

[source]
----
CURL(URL[|POST-Daten])
----

Lädt eine Webseite unter der angegebenen `URL` mit
HTTP `GET`. Wenn `POST-Daten` angegeben sind, werden
diese Daten mit `POST` gesendet. Gibt die Seite als String
zurück.

[source]
----
; Die Seite unter der URL http://beispiel.de/seite.php?id=1&action=view abrufen:
exten => 123,1,Set(foo=${CURL(http://beispiel.de/seite.php?id=1&action=view)})
----

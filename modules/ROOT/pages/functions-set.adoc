[[functions-set]]
= `SET()`

[source]
----
SET(Variablenname=Ausdruck)
----

Kann innerhalb von verschachtelten Ausdrücken verwendet werden, um
Variablen auf einen Wert zu setzen. (Verwechseln Sie diese Funktion nicht
mit der Applikation `Set()`!)

[source]
----
; ${a}, ${b}, ${c}, und ${d} auf 8 setzen:
exten => 123,1,Set(a=${SET(b=${SET(c=${SET(d=8)})})})

; es ist aber meist vorteilhafter, zugunsten der Übersichtlichkeit
; ein oder zwei Zeilen mehr zu schreiben
----

.Asterisk-Versionen:
————————| *1.2* |————————| *1.4* |————————| *1.6* |————————

-= Info about function 'SET' =-

[Synopsis]

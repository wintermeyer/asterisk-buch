[[agi-command-asyncagi-break]]
= `ASYNCAGI BREAK`

[source]
----
ASYNCAGI BREAK
----

Beendet die asynchrone Ausführung des AGI-Skripts.

.Asterisk-Versionen:
| | | | | *1.6* |————————

.Interner Hilfetext zu diesem AGI-Befehl in Asterisk 1.6:
Usage: ASYNCAGI BREAK
Break the Async AGI loop.
Runs Dead : No

.Differenz des internen Hilfetexts von Asterisk 1.4 zu 1.6:
— in Asterisk 1.4 nicht vorhanden —

[[functions-len]]
= `LEN()`

[source]
----
LEN(String)
----

Gibt die Länge (die Anzahl der Zeichen bzw. genauer gesagt: der Bytes)
eines Strings zurück.

[source]
----
; angenommen, ${test} ist "Hallo Welt"
exten => 123,1,Set(foo=${LEN(${test})})
; dann wird 10 zurückgegeben
----

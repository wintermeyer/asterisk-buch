[[applications-hangup]]
= `Hangup()`

Legt den aktuellen Channel auf.

[source]
----
Hangup()
----

Legt den aktuellen Channel bedingungslos auf.

Liefert stets -1 zurück.

[source]
----
exten => 123,1,Answer()
exten => 123,n,Playback(vm-goodbye)
exten => 123,n,Hangup()
----

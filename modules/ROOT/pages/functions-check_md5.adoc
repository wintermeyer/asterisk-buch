[[functions-check_md5]]
= `CHECK_MD5()`

[source]
----
CHECK_MD5(md5-string,string)
----

Die Funktion berechnet die MD5-Prüfsumme des 2. Parameters und
vergleicht diese mit der Prüfsumme, die im 1. Parameter übergeben wurde.
Wenn sie übereinstimmen, gibt die Funktion 1 zurück, sonst 0.

[source]
----
; Prüfe das Passwort anhand einer Checksumme
exten => 123,1,GotoIf(${CHECK_MD5(${MD5},${PASS})}?ok,error)
    
----

[[applications-endwhile]]
= `EndWhile()`

Beendet eine While-Schleife.

[source]
----
EndWhile()
----

Kehrt zur vorher aufgerufenen `While()`-Anwendung
zurück. Für eine vollständige Beschreibung dazu, wie eine While-Schleife
benutzt wird, siehe xref:applications-while.adoc[Abschnitt C.195, „`While()`“].

[source]
----
exten => 123,1,Answer()
exten => 123,n,Set(i=1)
exten => 123,n,While($[${i} < 5])
exten => 123,n,SayNumber(${i})
exten => 123,n,Set(i=$[${i} + 1])
exten => 123,n,EndWhile()
exten => 123,n,Hangup()
----

[[konferenzraum-definition]]
= Definition von Konferenzräumen

In Asterisk finden Konferenzen immer in virtuellen Konferenzräumen
statt. Diese müssen in der Konfigurationsdatei
`/etc/asterisk/meetme.conf` definiert werden. Dies
geschieht im Bereich `[rooms]`, und die Syntax ist denkbar
einfach: `conf =>
Konferenzraumnummer[,PIN][,Admin-PIN].`

Möchten Sie einen für jeden frei zugänglichen Konferenzraum mit der
Nummer 1234 einrichten, so muss der entsprechende Teil in der
`meetme.conf` wie folgt aussehen:

[source]
----
[rooms]
conf => 1234
----

Sollen zusätzlich noch die Räume 700 bis 705 mit jeweils dem
gleichen PIN 5678 eingerichtet werden, so sieht die `meetme.conf`wie folgt aus:

[source]
----
[rooms]
conf => 1234
conf => 700,5678
conf => 701,5678
conf => 702,5678
conf => 703,5678
conf => 704,5678
conf => 705,5678
----
